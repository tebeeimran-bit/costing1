# Tema Improvement
Membuat digitalisasi untuk New Project Costing.

## Definisi Masalah
Proses New Project Costing saat ini belum terdigitalisasi secara end-to-end, sehingga alur lintas fungsi (Engineering, Costing, Marketing) belum real-time, sulit ditelusuri, dan berisiko menyebabkan keterlambatan keputusan bisnis.

Masalah utama yang terjadi:
- Data project tersebar di beberapa media (file, chat, email, input manual).
- Status dokumen tidak selalu ter-update secara konsisten.
- Revisi dokumen sulit ditelusuri dengan cepat dan akurat.
- Risiko salah input dan duplikasi data masih tinggi.
- Monitoring progres project belum sepenuhnya berbasis indikator terukur.

## Analisa Kondisi Saat Ini
### Analisa Proses
- Input data awal project belum sepenuhnya standar, sehingga kualitas data berbeda antar user.
- Alur update dokumen masih bergantung komunikasi personal.
- Riwayat perubahan file dan versi belum selalu terdokumentasi rapi.
- Waktu pencarian data historis masih memakan waktu.
- Kontrol SLA proses costing belum terlihat jelas per tahapan.

### Analisa Akar Masalah
- Belum ada single source of truth untuk data New Project Costing.
- Belum ada workflow digital yang memaksa urutan proses dan validasi mandatory field.
- Integrasi antar tahapan (receipt, revisi, update, submit COGM) belum dikendalikan dengan KPI harian.
- Pelaporan masih cenderung reaktif, bukan proaktif.

## Rencana Improvement
### Strategi Utama
- Standarisasi data master dan form New Project.
- Penerapan workflow terstruktur dari penerimaan dokumen sampai submit COGM.
- Penerapan versioning dan history tracking untuk semua revisi dokumen.
- Penerapan validasi file dan mandatory field untuk menurunkan error input.
- Pembuatan dashboard monitoring status project dan bottleneck proses.
- Penerapan notifikasi untuk status kritis (pending input, pending pricing, overdue).
- Penyusunan SOP operasional dan pelatihan user lintas departemen.

### Tahapan Implementasi
1. Fase 1: Mapping proses As-Is dan penetapan To-Be Flow.
2. Fase 2: Implementasi form dan tracking digital inti.
3. Fase 3: Implementasi monitoring, KPI, dan notifikasi.
4. Fase 4: Stabilization, evaluasi, dan continuous improvement.

## Target Benefit
### Benefit Bisnis
- Lead time proses New Project Costing lebih cepat.
- Akurasi data meningkat karena validasi terstandar.
- Transparansi status project meningkat untuk seluruh stakeholder.
- Risiko kehilangan histori revisi dan file menurun signifikan.
- Kecepatan keputusan pricing dan submit ke Marketing meningkat.

### Target KPI (Terukur)
- Penurunan lead time end-to-end minimal 30 persen dalam 3 sampai 6 bulan.
- Penurunan error input dan duplikasi data minimal 50 persen.
- Kepatuhan update status harian di atas 95 persen.
- Waktu pencarian histori project kurang dari 5 menit per project.
- On-time submit COGM ke Marketing di atas 95 persen.

## Ringkasan Kalimat SPOK (Untuk Tema)
Perusahaan membuat digitalisasi proses New Project Costing untuk meningkatkan kecepatan, akurasi, dan transparansi pengendalian project.
